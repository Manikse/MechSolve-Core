# MechSolve-Core ⚙️

Tired of manually calculating support reactions and bending moments before every Strength of Materials assignment? This CLI script does it for you in a fraction of a second.

[Insert image of simply supported beam bending moment diagram]

Formula used for max moment: $M_{max} = \frac{F \cdot a \cdot b}{L}$

## Usage
Run the script from your terminal:
`python beam_calc.py -l 5 -f 10 -d 2`

**Output:** Instant calculation of support reactions and max bending moment. Save your time for actual engineering, not punching calculator buttons.
